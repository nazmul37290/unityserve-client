# UNITYSERVE

Live website link [https://unityserve-ee43d.web.app/](https://unityserve-ee43d.web.app/)

## overview
It is a website for volunteer management . where a user can post for need volunteer for any social events.
Any users can also participate in a event as a volunteer by requesting in a volunteer post.

## Features

- Add Post for need volunteer
- Request for be a volunteer
- Manage Your posts
- Toggle grid and table format for all posts
- Login with email and password
- Login with google
- Register for user

## Technologies used
- html
- react
- express
- node js
- mongodb
- tailwind css
- other npm packages

## How to run this repo locally

- install node js 
- add this command to project folder "npm i"
- then add another command "npm run dev"
